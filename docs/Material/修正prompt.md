## 三、内容创作与审查最终原则

### 核心原则

1. **全球视角优先原则**：任何历史叙述必须至少涵盖4个及以上不同区域的文明，确保非西方内容占比不低于40%

2. **多元因果原则**：历史事件的解释必须考虑多重因素（经济、政治、文化、技术、环境等），避免单一因果归因

3. **适度简化原则**：在保持准确性的前提下进行简化，简化必须明确标注，且不能导致根本性误解

4. **结构一致性原则**：各章节保持统一的框架结构，包括概述、案例、反思问题和现代联系

### 审查标准

| 维度 | 通过标准 |
|-----|---------|
| 时间准确性 | 时间线无错误，边界清晰，过渡自然 |
| 地域平衡 | 非西方文明内容占比不低于40%，涵盖非洲、美洲、亚洲、大洋洲 |
| 表述准确性 | 避免绝对化表述，使用适当限定词，体现历史解释的多元性 |
| 教育适配 | 零基础用户能够理解，知识结构循序渐进，包含实践指导 |
| 内容原创 | 避免过度模板化，各章节有独特价值，无重复内容 |
| 结构完整 | 各时代章节包含概述、关键变化、关键词、反思问题、案例和现代联系 |

### 内容扩展指导

1. **新增时代章节**：必须遵循现有框架结构，增加独特案例和多元视角
2. **案例选择**：优先选择具有全球影响力或代表性的事件和人物，涵盖不同地区
3. **语言风格**：保持通俗易懂，避免学术术语堆砌，必要术语需附带解释
4. **反思问题设计**：引导用户思考而非简单回忆，鼓励批判性思维和跨时代连接

### 定期审查机制

- 每季度进行一次内容一致性审查
- 新增内容需经过至少两名历史领域专家审核
- 用户反馈需定期收集并纳入内容改进
- 定期评估非西方内容比例是否达标
